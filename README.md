# 🎬 Netflix Data Analysis

This project analyzes Netflix dataset using Python to extract meaningful insights.

## 📊 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📁 Dataset
Netflix dataset containing movies and TV shows information.

## 🔍 Key Analysis
- Movies vs TV Shows distribution
- Top countries producing content
- Year-wise content growth
- Content distribution (Pie Chart)

## 📈 Insights
- Majority of content on Netflix is Movies
- USA has the highest number of shows
- Content increased significantly after 2015
- Drama and International content are popular
